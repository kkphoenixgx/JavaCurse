- Encapsulamento

  Encapsulamento é basicamente esconder detalhes de 
implementação de uma classe, controlando o que pode
ser acessado e o que não pode, mantendo o objeto em
um estado consistente.
  
  Mantendo os atributos privados usando private e somente
podendo ser acessados com métodos gets and setters:

   Para pegar o atributo:
ex: public y getX(return X){ return X} sendo X o nome do atributo
e y o tipo do dado do atributo.

   Para setar o atributo:
public void setX(y X){ this.X = X } sendo X o nome da
variavel e y o tipo do dado.

JavaBeans = É um padrão(convenção) utilizado pelos 
programadores Java para projetos.
